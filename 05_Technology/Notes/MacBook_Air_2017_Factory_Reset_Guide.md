# MacBook Air 2017 (Intel) 双系统恢复出厂设置完整教程

本教程适用于安装了 Windows + macOS 双系统的 2017 年 MacBook Air (Intel 芯片)，指导你彻底删除所有数据并恢复到出厂状态。

## 准备工作

### 必需条件

- **稳定的网络连接**（Wi-Fi 或以太网）- 用于下载 macOS
- **电源适配器** - 确保全程接入电源
- **Apple ID 密码** - 可能需要用于关闭"查找我的 Mac"
- **足够的时间** - 整个过程约需 1-3 小时

### 检查清单

- [ ] 确认已备份所有重要数据
- [ ] 记住 Apple ID 和密码
- [ ] 确保电脑已接入电源
- [ ] 确保网络连接稳定

## 备份重要数据

### macOS 数据备份

1. **使用 Time Machine**
   - 连接外置硬盘
   - 系统偏好设置 → Time Machine → 选择备份磁盘
   - 点击"立即备份"

2. **手动备份重要文件**
   - 文档、照片、音乐、下载文件夹
   - 桌面文件
   - 浏览器书签和密码
   - 应用程序的配置文件

### Windows 数据备份

1. 重启进入 Windows（开机时按住 `Option` 键选择 Windows）
2. 将重要文件复制到外置硬盘或云存储
3. 导出浏览器书签、密码等

### 退出相关账户

在 macOS 中：
1. **退出 iCloud**：系统偏好设置 → Apple ID → 概览 → 退出登录
2. **关闭"查找我的 Mac"**：系统偏好设置 → Apple ID → iCloud → 取消勾选"查找我的 Mac"
3. **退出 iMessage**：打开信息 App → 信息菜单 → 偏好设置 → iMessage → 退出登录
4. **取消授权 iTunes**：打开音乐/iTunes → 账户 → 授权 → 取消对这台电脑的授权

## 删除 Windows 分区（Boot Camp）

> **注意**：如果你打算直接抹掉整个磁盘，可以跳过这一步，直接进入下一节。

### 方法一：使用 Boot Camp 助理（推荐）

1. 启动进入 macOS
2. 打开 **启动台** → **其他** → **Boot Camp 助理**
3. 点击 **继续**
4. 选择 **移除 Windows 10 或更高版本**
5. 点击 **恢复** - 这会删除 Windows 分区并将空间还给 macOS

### 方法二：使用磁盘工具

如果 Boot Camp 助理无法正常工作：

1. 打开 **磁盘工具**（启动台 → 其他 → 磁盘工具）
2. 在左侧选择 **BOOTCAMP** 分区
3. 点击 **抹掉**，格式选择 **Mac OS 扩展（日志式）**
4. 打开 **终端**，运行以下命令合并分区：
   ```bash
   diskutil list  # 查看磁盘分区情况
   diskutil eraseVolume free free /dev/diskXsY  # X和Y替换为BOOTCAMP分区号
   diskutil apfs resizeContainer diskXsZ 0  # 扩展macOS容器
   ```

## 抹掉磁盘并重装 macOS

### 步骤一：进入 macOS 恢复模式

1. 完全关闭 Mac
2. 按下电源键，**立即同时按住** `Command (⌘) + R`
3. 持续按住直到看到 Apple 标志或旋转地球
4. 等待进入 **macOS 实用工具** 界面

#### 恢复模式选项说明

| 组合键 | 功能 |
|--------|------|
| `Command + R` | 重装当前系统版本 |
| `Option + Command + R` | 升级到最新兼容的 macOS |
| `Shift + Option + Command + R` | 重装出厂预装的 macOS |

> **建议**：如果要完全恢复出厂状态，使用 `Shift + Option + Command + R`

### 步骤二：连接网络

如果看到旋转地球或需要下载系统：
1. 点击右上角 Wi-Fi 图标
2. 选择你的 Wi-Fi 网络并输入密码

### 步骤三：抹掉磁盘

1. 在 macOS 实用工具窗口，选择 **磁盘工具**，点击 **继续**

2. 在磁盘工具中：
   - 点击左上角 **显示** → **显示所有设备**
   - 在左侧边栏选择**最顶层的物理磁盘**（通常叫 "APPLE SSD..." 或 "Internal"）
   - **不要选择下面的分区！**

3. 点击顶部的 **抹掉** 按钮

4. 设置以下选项：
   - **名称**：`Macintosh HD`（或你喜欢的名字）
   - **格式**：`APFS`（如果是 macOS High Sierra 或更高）或 `Mac OS 扩展（日志式）`
   - **方案**：`GUID 分区图`

5. 点击 **抹掉**，等待完成

6. 完成后点击 **完成**，然后关闭磁盘工具

### 步骤四：重新安装 macOS

1. 返回 macOS 实用工具窗口
2. 选择 **重新安装 macOS**，点击 **继续**
3. 按照屏幕提示操作：
   - 同意许可协议
   - 选择刚才抹掉的磁盘（Macintosh HD）
   - 点击 **安装**

4. 等待安装完成（可能需要 30 分钟到 2 小时，取决于网速）

5. 安装期间电脑会自动重启多次，**不要中断**

### 步骤五：完成设置

安装完成后，Mac 会显示设置助理：

**如果你要继续使用这台 Mac：**
- 按照提示完成初始设置（语言、Apple ID、Wi-Fi 等）

**如果你要出售或转让这台 Mac：**
- 按 `Command + Q` 退出设置助理
- Mac 将保持在"开箱即用"状态，新主人可以自行设置

## 常见问题解决

### 问题 1：无法进入恢复模式

**解决方案：**
1. 确保完全关机后再尝试
2. 按下电源键后**立即**按住组合键
3. 尝试使用网络恢复：`Option + Command + R`
4. 重置 NVRAM：关机后按 `Option + Command + P + R` 约 20 秒

### 问题 2：磁盘工具看不到磁盘

**解决方案：**
1. 点击 **显示** → **显示所有设备**
2. 如果仍看不到，可能是硬件问题，需要联系 Apple 支持

### 问题 3：抹掉磁盘时报错

**解决方案：**
1. 尝试先点击 **急救** 修复磁盘
2. 使用终端强制抹掉：
   ```bash
   diskutil list
   diskutil eraseDisk APFS "Macintosh HD" GPT /dev/disk0
   ```

### 问题 4：系统下载失败

**解决方案：**
1. 检查网络连接
2. 尝试使用有线网络（需要 USB 以太网适配器）
3. 更换 Wi-Fi 网络（避免使用需要网页认证的网络）
4. 稍后再试（可能是 Apple 服务器繁忙）

### 问题 5：提示"无法验证 macOS"

**解决方案：**
1. 检查日期和时间设置是否正确
2. 在恢复模式下打开终端，运行：
   ```bash
   ntpdate -u time.apple.com
   ```
3. 或手动设置日期：
   ```bash
   date 0126120026  # 格式：月日时分年（最后两位）
   ```

### 问题 6：忘记关闭"查找我的 Mac"

如果已经抹掉磁盘但忘记关闭：
1. 在另一台设备上访问 [iCloud.com](https://www.icloud.com)
2. 登录你的 Apple ID
3. 点击 **查找我的** → 选择这台 Mac → **从账户中移除**
