# 基础课程：信息感知体系重塑——Obsidian + Gemini Scribe + Simple RSS 实战指南

## 1. 课程概述 (Course Overview)

**核心目标**：帮助用户从零开始构建一套基于 AI 的本地化信息获取与知识内化系统。不仅教授工具的使用，更强调“主动摄取”与“结构化沉淀”的方法论。

**适用人群**：

- **知识工作者**：需要高效处理大量行业资讯。
- **研究人员**：需要追踪特定领域的最新动态（arXiv, GitHub）。
- **内容创作者**：建立自动化的素材收集库。
- **Obsidian 用户**：希望打造 All-in-One 的 Second Brain。

**预备知识**：

- 基本的 Obsidian 操作（Markdown 语法，插件安装）。
- 对 RSS 概念有初步了解。
- (可选) Google Gemini API Key 获取能力。

---

## 2. 课程大纲 (Curriculum Outline)

本课程分为三个阶段：**基础搭建 (Foundation)** -> **工作流实战 (Workflow)** -> **进阶定制 (Advanced)**。

### 阶段一：理念与基础搭建 (Phase 1: Foundation)

**目标**：理解“为什么”并完成环境配置。

#### Lesson 1.1: 信息感知体系的进化 (Evolution of Information Perception)

- **概念解析**：
  - 传统信息流的痛点：算法黑箱、噪音过大、被动接收。
  - RSS 的复兴：主动订阅、去中心化、无算法干扰。
  - AI 的赋能：将“阅读量”转化为“信息密度”，解决 RSS 信息过载问题。
- **核心理念**：Input (RSS) -> Process (AI Summarization) -> Output (Knowledge Graph).

#### Lesson 1.2: 环境准备与工具安装 (Setup & Installation)

- **Obsidian 基础设置**：
  - 推荐的文件夹结构 (`00_Inbox`, `00_RSS_News`, `Daily_Brief`).
- **Simple RSS 插件安装与配置**：
  - 安装来源（第三方插件市场 / GitHub Release）。
  - **关键配置详解**：
    - OPML 导入流程（含 URL 校验功能演示）。
    - 文件名格式化 (`{{date}}-{{title}}`) 的重要性。
    - 模板设置 (Frontmatter, Tags)。

#### Lesson 1.3: Gemini Scribe 插件配置 (AI Setup)

- **Gemini Scribe 简介**：
  - 什么是 Gemini Scribe？(Obsidian 内的 AI Agent 运行环境)。
- **模型配置**：
  - 获取 Google Gemini API Key。
  - 配置 Model (Gemini Pro / Flash) 以平衡速度与成本。
- **Prompts 库导入**：
  - 介绍 `gemini-scribe/Prompts` 目录结构。
  - 导入 `daily-brief-expert.md` 等核心 Prompt。

---

### 阶段二：工作流实战 (Phase 2: Workflow)

**目标**：掌握每日“收集-清洗-内化”的标准作业程序 (SOP)。

#### Lesson 2.1: 订阅源管理与 OPML 迁移 (Feed Management)

- **寻找优质源**：
  - 推荐工具：RSSHub, Politepol。
  - 筛选原则：信噪比、更新频率、原创性。
- **OPML 实战**：
  - 从 Feedly/Inoreader 导出。
  - 使用 Simple RSS 的 **Validate & Import** 功能清洗无效源。
  - 建立分类体系（如：`Tech`, `Finance`, `Design`）。

#### Lesson 2.2: 每日简报生成 (Daily Briefing)

- **自动化收集**：
  - 演示 Simple RSS 的一键同步。
  - 观察 `00_RSS_News` 下生成的文件结构（按日期自动排序带来的便利）。
- **AI 智能摘要**：
  - 运行 Gemini Scribe `Daily Brief` Agent。
  - **Prompt 解析**：解读 `daily-brief-expert.md` 如何让 AI 按主题分类、提炼要点。
  - **生成结果分析**：查看生成的 `YYYY-MM-DD_Daily_Brief.md`。

#### Lesson 2.3: 深度阅读与知识连接 (Deep Read & Link)

- **从简报到原文**：
  - 通过简报中的链接跳转回本地 RSS Markdown 文件。
  - 使用 Obsidian 双链 (`[[]]`) 关联到现有知识库。
- **渐进式归档**：
  - 什么样的内容值得精读？
  - 如何将 RSS 笔记转化为永久笔记 (Permanent Note)。

---

### 阶段三：进阶定制与维护 (Phase 3: Advanced)

**目标**：根据个人需求定制系统。

#### Lesson 3.1: Prompt Engineering 定制 (Customizing Prompts)

- **修改 `daily-brief-expert.md`**：
  - 调整关注点（例如：只关注 AI 编程领域的更新）。
  - 调整输出格式（表格 vs 列表）。
  - 增加多语言翻译（如：英文源自动生成中文摘要）。

#### Lesson 3.2: 自动化与脚本扩展 (Automation)

- **高级过滤**：
  - 在 Simple RSS 中使用正则过滤标题。
- **QuickAdd / Templater 联动**：
  - 自动将每日简报移动到归档目录。
  - 生成周报/月报 (利用 `monthly-brief-expert.md`)。

#### Lesson 3.3: 维护与优化 (Maintenance)

- **定期清理**：
  - 设置自动删除过期的 RSS 文件（保持仓库轻量）。
- **Prompt 迭代**：
  - 随着模型升级（如 Gemini 1.5 Pro），优化 Context Window 的使用。

## 5. 附录：前置条件与资源 (Appendix)

### 5.1 基础环境准备

在开始课程前，请确保已准备好以下环境：

1. **GitHub 账号**
    - 主要用于下载插件 Release 包和后续可能的版本控制。
    - [注册地址](https://github.com/join)
2. **Git (可选，但在 Mac 上推荐)**
    - 用于版本管理和高级插件安装。
    - 检查是否安装：在终端输入 `git --version`
    - 安装方法：终端输入 `xcode-select --install` 或通过 [Homebrew](https://brew.sh/) 安装 `brew install git`。
3. **Obsidian**
    - 知识库核心载体。
    - [下载地址](https://obsidian.md/)
4. **Google Gemini API Key**
    - 用于驱动 Gemini Scribe。
    - [申请地址](https://aistudio.google.com/app/apikey)

### 5.2 核心插件安装

本课程依赖以下插件，请按需下载：

#### **1. Gemini Scribe (AI 助手)**

* **功能**：Obsidian 内的 AI Agent 运行环境，支持读取本地文件。
- **安装**：在 Obsidian 设置 -> 第三方插件 -> 社区插件市场搜索 "Gemini Scribe" 安装。
- **配置**：填入 API Key，并将本课程提供的 `Prompts` 文件夹复制到插件配置目录。
	- 前往 Google AI Studio 申请 API KEY：https://aistudio.google.com/

#### **2. Simple RSS (自定义增强版)**

* **功能**：支持 URL 校验和日期前缀的 RSS 管理器。
- **下载地址**：[RayLee1997/obsidian-simple-rss Releases]([https://github.com/RayLee1997/obsidian-simple-rss/releases](https://github.com/RayLee1997/obsidian-simple-rss/tree/master/release))
	- https://github.com/RayLee1997/obsidian-simple-rss/tree/master/release
- **安装方法**：下载最新版本的 zip 包，解压放入 `.obsidian/plugins/simple-rss/` 文件夹。
- **推荐RSS订阅源**：https://github.com/RayLee1997/obsidian-simple-rss/blob/master/release/rss/rss_subscriptions_important.opml

#### **3. Markdown to Stat (公众号发布工具)**

* **功能**：将 Markdown 笔记一键转换为微信公众号格式，或导出高清长图。
- **下载地址**：[RayLee1997/obsidian-md2stat Releases]([https://github.com/RayLee1997/obsidian-simple-rss/tree/master/release](https://github.com/RayLee1997/obsidian-md2stat/tree/main/releases))
	- https://github.com/RayLee1997/obsidian-md2stat/tree/main/releases
- **安装方法**：下载最新版本的 zip 包，解压放入 `.obsidian/plugins/obsidian-md2stat/` 文件夹。
