#OpenCode与Obsidian

1. 终于找到了一款满意的 Notes 替代，就是 Obsidian，之前从 Notes 迁移到 Notion，Notion 本身没什么不好的，只是 Obsidian 更像是简易版的 vs code，Gemmi Scribe 插件做的也很好，工作界面布局很符合我写东西的习惯；
2. Claude Code 暂时解决不了账户支付的问题，被迫尝试了一下 OpenCode + OpenRouter 的开源路线，不过总算用上了 Claude Opus 4.5 和 Gemini 3 Pro ，支持支付宝和微信预付费，按 tokens 收费，成本其实是有点高的。比如我本地一个不算太大的 Code base 初始化一下，使用 Claude Opus 4.5 整整花了 $5 多，对比 Gemini 3 Pro 贵了差不多 90%，后面尝试下 OpenCode + Github Copilot 联合订阅模式，月供 $39 应该是个非常不错的选择；
3. 这套组合基本上解决了本地知识管理 + 思路整理、方案规划 + 本地代码快速工具集成的问题，可能这也代表了一种趋势，一个想法和 AI 开完会后转交 Code Agents 完成一套定制系统，用完即弃；
