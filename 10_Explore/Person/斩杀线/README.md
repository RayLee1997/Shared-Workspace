# 斩杀线事件（牢A / 斯奎奇大王）资料整理

本目录用于整理“美国斩杀线”概念在中文互联网的传播脉络，以及与之相关的网络人物（主要为“牢A / 斯奎奇大王”）与争议点。内容以可公开检索到的媒体文章/平台内容为依据；对人物身份与具体故事细节不做无证断言。

## 使用边界（重要）

- 本目录不收集/不记录住址、电话、证件号等敏感个人信息；不转载开盒信息。
- 将信息分为：可核对事实 / 当事人自述 / 媒体转述 / 评论推测；每条尽量给出来源。
- 本目录的目标是“可追溯的整理”，不是“定性审判”。

## 可靠性标注（建议口径）

- 高：一手材料（原视频/原帖/当事人公开账号原文）、多家独立来源一致可交叉验证。
- 中：主流媒体报道或平台文章，但包含转述/评论，细节仍需核验。
- 低：自媒体号、强立场文章、难以核验的轶事汇编、匿名整理站点。
