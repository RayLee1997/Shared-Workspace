> **更新内容**: 深度集成 **IMF Data MCP (宏观数据)** 与 **Brave Search MCP (实时资讯)** 数据源，细化数据获取路径。

---

## 一、 调研背景与目标

**调研背景**：2025年是中国“十四五”规划的收官之年，也是全球供应链重构的关键节点。回顾1995年至今的30年，涵盖了中国加入WTO前后的巨变、2008年金融危机后的调整以及近年来“双循环”战略的实施。

**核心目标**：

1. **复盘现状**：精准复盘2025年中国外贸的整体表现（顺差、规模），剔除汇率扰动。
2. **量化趋势**：利用IMF长周期数据，量化分析1995-2025年进口贸易金额（美元计价）的长期趋势。
3. **结构解构**：深度解构30年间进出口商品结构与市场结构的根本性变化。
4. **宏观影响**：分析外汇盈余的来源构成及其对宏观经济的影响。

## 二、 核心调研模块与数据获取策略

### 模块一：2025年进出口贸易全景（截面分析）

**目标**：确立2025年当年的贸易基准数据，作为分析的终点。

#### 1. 总量指标与贸易平衡

- **核心数据**：进出口总值 (USD)、贸易顺差 (USD)。
- **统一口径**：全篇分析统一采用 **美元 (USD)** 计价的进出口金额作为衡量贸易规模的核心指标。
- **MCP 数据源**：
  - `imf-data-mcp` -> `fetch_dot_data`: 获取中国对全球（World, `W00`）的出口（`TXG_FOB_USD`）和进口（`TMG_CIF_USD`）数据。
  - `imf-data-mcp` -> `fetch_bop_data`: 获取2025年季度国际收支平衡表，特别是经常账户（`BCA_BP6_USD`）和服务贸易差额。
  - `imf-data-mcp` -> `fetch_ifs_data`: 获取2025年人民币对美元汇率（`ENDA_XDC_USD_RATE`），用于汇率折算。

#### 2. 贸易伙伴结构（2025截面）

- **核心关注**：对美、对欧、对东盟、对“一带一路”国家的贸易占比。
- **MCP 数据源**：
  - `imf-data-mcp` -> `fetch_dot_data`: 获取中国对 `US` (美国), `U2` (欧元区), `ASEAN` (东盟主要国家: `ID+TH+VN+MY+PH+SG+...`) 的双边贸易数据。
  - `brave-search-mcp` -> `brave_news_search`: 搜索 "China ASEAN trade 2025", "China EU trade friction 2025" 获取最新的贸易谈判和非关税壁垒动态。

#### 3. 经营主体与新业态

- **核心关注**：民营企业占比、跨境电商增速。
- **MCP 数据源**：
  - `brave-search-mcp` -> `brave_web_search`: 搜索 "2025年中国跨境电商进出口规模", "民营企业进出口占比 2025 海关总署"。
  - `brave-search-mcp` -> `brave_summarizer`: 对搜索到的海关总署月度报告进行摘要，提取关键增长数据。

### 模块二：1995-2025年进口贸易金额趋势分析（纵向分析）

**目标**：分析30年长周期趋势，从“世界工厂”向“世界市场”转变。

#### 1. 进口规模演变（30年长周期）

- **数据点**：1995-2025历年进口总额 (CAGR)。
- **MCP 数据源**：
  - `imf-data-mcp` -> `fetch_dot_data`: `freq="A"`, `country="CN"`, `indicator="TMG_CIF_USD"`, `start=1995`, `end=2025`。
  - `imf-data-mcp` -> `fetch_ifs_data`: 获取同期 GDP 数据 (`NGDP_USD`)，计算 **进口渗透率** (Imports/GDP)。

#### 2. 关键历史节点复盘

- **关注点**：入世(2001)、金融危机(2008)、贸易战(2018)、疫情后(2023-2025)。
- **MCP 数据源**：
  - `imf-data-mcp` -> `fetch_dot_data`: 分段计算上述区间的 CAGR。
  - `brave-search-mcp` -> `brave_web_search`: 搜索特定年份的政策背景，如 "China WTO accession tariff reduction impact report"。

#### 3. 重点商品进口金额分析

- **核心关注**：原油、铁矿石、芯片等战略物资的进口金额走势。
- **MCP 数据源**：
  - `brave-search-mcp` -> `brave_web_search`: 搜索 "China crude oil import volume vs price 2025", "China semiconductor import value 2025 trend"。
  - `imf-data-mcp` -> `fetch_ifs_data`: 获取全球大宗商品价格指数 (Commodity Price Index, `PALLFNF_INDEX`) 作为价格参照。

---

### 模块三：进出口结构的深层演变（1995-2025）

**目标**：捕捉产业升级轨迹。

#### 1. 出口结构：从旧三样到新三样

- **分析维度**：劳动密集型 vs 技术密集型。
- **MCP 数据源**：
  - `brave-search-mcp` -> `brave_web_search`: 搜索 "China exports new three (EV, battery, solar) 2025 growth", "high-tech product export share China 1995-2025"。
  - `imf-data-mcp` -> `fetch_dot_data`: 对比中国对发达国家（US/EU/JP）与发展中国家（Africa/LatAm）的出口增速差异，验证市场多元化策略。

#### 2. 进口结构：替代与升级

- **分析维度**：中间品国产化替代 vs 消费品进口升级。
- **MCP 数据源**：
  - `brave-search-mcp` -> `brave_web_search`: 搜索 "China intermediate goods import substitution 2025", "luxury goods import growth China 2025"。

---

### 模块四：外汇盈余与国际收支分析

**目标**：分析巨额顺差的去向与影响。

#### 1. 经常账户与国际收支

- **核心数据**：经常账户余额、资本与金融账户净额。
- **MCP 数据源**：
  - `imf-data-mcp` -> `fetch_bop_data`:
    - `BCA_BP6_USD` (经常账户余额)
    - `BGS_BP6_USD` (货物与服务差额)
    - `BF_BP6_USD` (金融账户净额)
    - `start=2015`, `end=2025` (重点关注近10年变化)。

#### 2. 外汇储备与汇率

- **核心数据**：外汇储备规模、人民币汇率波动。
- **MCP 数据源**：
  - `imf-data-mcp` -> `fetch_ifs_data`:
    - `RAFA_USD` (外汇储备，美元计价)
    - `ENDA_XDC_USD_RATE` (期末汇率)
  - `brave-search-mcp` -> `brave_web_search`: 搜索 "China foreign reserves composition change 2025", "PBOC gold reserves 2025"。

## 三、 补充建议：需要特别关注的“隐形”数据

1. **增加值贸易 (TiVA)**：
    - 使用 `brave-search-mcp` 搜索 OECD TiVA 数据库的最新报告（如有2025更新），对比 Gross Exports 和 Value Added Exports 的差异。
2. **转口贸易挤水分**：
    - 使用 `imf-data-mcp` -> `fetch_dot_data` 对比 "China -> HK Export" 与 "HK -> China Import" 的数据镜像差异。
    - 数据代码：`CN` 对 `HK` 的 `TXG_FOB_USD` vs `HK` 对 `CN` 的 `TMG_CIF_USD`。
