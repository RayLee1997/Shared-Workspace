# Windows 纯界面操作翻墙方案（零配置）

## 一、最简单方案：Clash Verge（推荐）

### 为什么选择 Clash Verge？

- ✅ 完全图形界面，点点鼠鼠就能用
- ✅ TUN 模式一键开启
- ✅ 自动安装驱动
- ✅ 无需任何配置文件编辑

---

## 二、完整操作步骤（5 分钟搞定）

### 第一步：获取订阅链接（墙内完成）

#### 推荐机场（墙内可直接访问）

**方案 A：奈云（最简单）**

```
1. 百度搜索："奈云机场" 或 "奈云加速器"
2. 找到官网镜像站（通常是 .com 或 .net 域名）
3. 点击"注册"
4. 填写邮箱（随便一个邮箱即可）
5. 设置密码
6. 点击"购买套餐" → 选择月付（15-20元）
7. 支付宝付款
8. 进入"用户中心" → "我的订阅"
9. 找到"Clash 订阅" → 点击"复制订阅链接"
```

**方案 B：TAG 机场**

```
1. 搜索引擎搜："TAG机场官网"
2. 找到最新镜像域名
3. 注册账号（邮箱即可）
4. 购买套餐（月付15元起）
5. 复制 Clash 订阅链接
```

**方案 C：WgetCloud（机场推荐列表）**

```
1. 访问：https://github.com/Alvin9999/new-pac/wiki
2. 这个页面墙内可访问
3. 里面有多个墙内可访问的机场链接
4. 选一个注册购买即可
```

### 第三步：安装 Clash Verge

```
纯界面操作，超级简单：

1. 双击下载的 .exe 文件
2. 如果弹出"Windows已保护你的电脑"
   → 点击"更多信息"
   → 点击"仍要运行"
3. 点击"安装"（默认安装到 C 盘即可）
4. 等待安装完成（约30秒）
5. 自动启动 Clash Verge

### 第四步：添加订阅（关键步骤）

```
1. 点击左侧 "订阅" 图标（📁）

2. 点击右上角 "+ 新建" 按钮

3. 在弹出窗口中：
   ┌────────────────────────────┐
   │ 新建订阅                    │
   ├────────────────────────────┤
   │ 类型： [URL ▼]             │
   │ 名称： 我的机场             │
   │ URL:  [粘贴订阅链接]        │
   │                            │
   │      [取消]  [保存]         │
   └────────────────────────────┘

4. 类型选择："URL"（默认就是）
5. 名称随便填（如：我的机场）
6. URL 框里粘贴刚才复制的订阅链接
7. 点击"保存"

8. 等待自动下载配置（3-5秒）
9. 看到订阅名称右边显示✅绿色对勾
```

---

### 第五步：开启 TUN 模式（核心）

```
1. 点击左侧 "设置" 图标（⚙️）

2. 找到 "TUN 模式" 区域
   ┌────────────────────────────┐
   │ TUN 模式                    │
   │ ○ 关闭  ○ 开启             │
   └────────────────────────────┘

3. 点击"开启"

4. 第一次会弹窗："需要安装 Wintun 驱动"
   → 点击"确定"

5. 弹出 Windows UAC 提示（用户账户控制）
   → 点击"是"（需要管理员权限）

6. 等待驱动安装（10-20秒）

7. 看到界面显示：
   ┌────────────────────────────┐
   │ TUN 模式                    │
   │ ○ 关闭  ● 开启  🟢已启用   │
   └────────────────────────────┘

8. 右下角系统托盘的 Clash 图标变为彩色
```

---

### 第六步：选择节点

```
1. 点击左侧 "代理" 图标（🌐）

2. 看到节点列表：
   ┌────────────────────────────┐
   │ PROXY                      │
   │  ○ 香港 01  12ms           │
   │  ○ 香港 02  15ms           │
   │  ○ 日本 01  45ms           │
   │  ○ 美国 01  180ms          │
   │  ...                       │
   │                            │
   │  [测速] [自动选择]          │
   └────────────────────────────┘

3. 点击"测速"按钮（等待5秒）

4. 延迟数字会更新，选择延迟最低的节点
   （推荐：香港、日本、新加坡，延迟 <100ms）

5. 点击该节点前面的圆圈 ○ 变成 ●

6. 完成！
```

---

### 第七步：验证是否成功

```
1. 打开浏览器（Chrome、Edge 都行）

2. 访问：https://www.google.com
   → 能打开就成功了！✅

3. 访问：https://twitter.com
   → 能打开就成功了！✅

4. 检查 IP：访问 https://ip.sb

## 四、常见问题（界面解决）

### 问题 1：TUN 驱动安装失败

**现象：**

```
点击开启 TUN 后，显示"驱动安装失败"
或者一直转圈不成功
```

**解决：**

```
方法 1：重新启动 Clash（最简单）
1. 右上角点击 ×关闭软件
2. 右键以"管理员身份运行" Clash Verge
3. 重新开启 TUN 模式

方法 2：手动下载驱动
1. 访问：https://www.wintun.net/
2. 下载 wintun.dll
3. 复制到：C:\Program Files\Clash Verge\
4. 重启 Clash

方法 3：关闭杀毒软件
1. 临时关闭 360、火绒等杀毒软件
2. 关闭 Windows Defender（系统设置→更新和安全→Windows安全中心）
3. 重新安装驱动
```

### 问题 2：订阅更新失败

**现象：**

```
点击更新订阅，显示"更新失败"或"连接超时"
```

**解决：**

```
方法 1：检查订阅链接
1. 回到机场网站
2. 重新复制订阅链接
3. 删除旧订阅，添加新的

方法 2：手动下载配置
1. 浏览器打开订阅链接
2. 会下载一个 .yaml 文件
3. Clash 中：订阅 → 新建 → 类型选"Local"
4. 选择刚下载的文件

方法 3：联系机场客服
1. 机场网站通常有客服 TG/邮箱
2. 说明情况，要备用订阅链接
```

### 问题 3：能连上但网速很慢

**现象：**

```
YouTube 看不了高清
Twitter 图片加载慢
```

**解决：**

```
1. 代理页面 → 点击"测速"
2. 换一个延迟更低的节点
3. 优先选择：
   - 香港节点（最快）
   - 日本节点
   - 新加坡节点
4. 避开美国、欧洲节点（太远）

5. 如果所有节点都慢：
   - 升级机场套餐（高速线路）
   - 或者换一个机场
```

### 问题 4：Google 能打开，Twitter 打不开

**现象：**

```
部分网站能访问，部分不能
```

**解决：**

```
1. 设置 → 代理模式 → 改为"全局"
   （默认是"规则"）

2. 或者清除浏览器缓存：
   Chrome: Ctrl + Shift + Delete
   选择"全部时间" → 清除数据

3. 刷新网页（Ctrl + F5 强制刷新）
```

### 问题 5：系统提示"网络异常"

**现象：**

```
QQ、微信等国内软件提示网络连接失败
```

**解决：**

```
1. 设置 → 代理模式 → 确认是"规则"模式
   （不要用"全局"）

2. 设置 → 允许局域网 → 关闭 [ ]

3. 重启国内软件（QQ、微信等）

原因：规则模式下国内流量会自动直连
```

## 六、节点优化（纯界面）

### 自动选择最快节点

```
1. 代理页面 → 找到"自动选择"选项

2. 点击启用：
   ┌────────────────────────────┐
   │ ● 自动选择                  │
   │   → 根据延迟自动切换节点    │
   └────────────────────────────┘

3. 设置测速间隔：
   - 每 5 分钟自动测一次速
   - 自动切换到最快节点
```

### 节点筛选

```
代理页面 → 搜索框

输入关键词筛选：
- "香港" → 只显示香港节点
- "日本" → 只显示日本节点
- "01" → 只显示 01 号节点
```
